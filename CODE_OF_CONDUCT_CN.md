# 行为准则

## 我们的承诺

PPT Master 致力于营造一个友好、尊重、专业的开源社区。

作为项目的维护者、贡献者和参与者，我们坚持让每个人都能在免受骚扰的环境下参与本项目，无论其年龄、体型、残疾、民族、性别特征、性别认同或表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、世系、肤色、宗教或性取向与性别认同。

我们的目标是构建一个能安心协作、开放交流、自由贡献的社区。

## 期望的行为

有助于营造积极社区氛围的行为包括：

- 待人友善，有同理心和耐心
- 尊重不同的意见和经验
- 用专业方式给予和接受建设性反馈
- 对自己的错误负责并从中学习
- 以项目和社区的利益为重

## 不可接受的行为

不可接受的行为包括：

- 人身攻击
- 骚扰、恐吓、威胁或跟踪他人
- 带性暗示的语言、图像或令人不适的性关注
- 引战、故意挑衅或反复恶意争论
- 未经许可发布他人隐私信息，包括物理地址或电子邮件地址
- 其他任何在专业社区环境中不宜出现行为

## 适用范围

本行为准则适用于所有项目空间，包括但不限于：

- GitHub issues
- Pull requests
- Discussions
- 项目文档
- 与项目相关的社区聊天或社交空间
- 代表项目的公开互动场合

## 举报

如果您遇到或目睹违反本准则的行为，请举报至：

**heyug3@gmail.com**

如有可能，请包含以下内容：

- 事件描述
- 发生地点
- 链接、截图或其他相关证据
- 有助于审查报告的其他背景信息

我们会尽可能迅速和公正地审查所有举报，努力保护举报人的隐私和安全。

## 执行措施

项目维护者负责解释和执行本行为准则。

对于违反本准则的行为，维护者可酌情采取任何措施，包括：

- 要求改正
- 发出警告
- 删除评论、issue、commit 或其他贡献
- 暂时限制参与项目
- 永久禁止参与项目空间

执行决策均以保护社区和维护健康的工作环境为目标。

## 执行原则

维护者会根据以下因素决定处理方式：

- 行为的严重程度
- 是否属于重复行为
- 行为是否故意
- 对个人和社区的影响

处理方式包括私下警告、临时封禁或永久封禁，视情节而定。

## 维护者责任

维护者应本着善意以统一标准执行本准则，并在必要时清楚说明处理决定的理由。最终目标始终是维护一个安全、受尊重、可协作的社区。

## 出处

本行为准则基于 Contributor Covenant 2.1 版的原则。官方 Contributor Covenant 资料和采纳指南可在 Contributor Covenant 网站上查看。
