# 像素复古模板 (Pixel Retro Style) - 设计规范

> 适用于技术分享、编程教学、游戏相关演示、极客风格内容展示等场景。

---

## 一、模板概述

| 属性           | 描述                                                   |
| -------------- | ------------------------------------------------------ |
| **模板名称**   | pixel_retro（像素复古模板）                            |
| **适用场景**   | 技术分享、编程教学、游戏介绍、极客风格展示             |
| **设计调性**   | 复古游戏、霓虹赛博、极客科技、8-bit 风格               |
| **主题模式**   | 暗色主题（深空黑背景 + 霓虹强调色）                    |

---

## 二、画布规范

| 属性           | 值                            |
| -------------- | ----------------------------- |
| **格式**       | 标准 16:9                     |
| **尺寸**       | 1280 × 720 px                 |
| **viewBox**    | `0 0 1280 720`                |
| **页面边距**   | 左右 60px，上 50px，下 40px   |
| **内容安全区** | x: 60-1220, y: 50-680         |

## 五、页面结构

### 通用布局

| 区域       | 位置/高度     | 内容说明                               |
| ---------- | ------------- | -------------------------------------- |
| **顶部**   | y=0, h=4-6px  | 霓虹绿装饰线（双线效果）               |
| **标题区** | y=50, h=70px  | 页面标题 + 英文副标题                  |
| **内容区** | y=130, h=510px| 主要内容区域                           |
| **页脚**   | y=680, h=40px | 页码、装饰线、进度指示                 |

### 装饰元素

- **顶部装饰线**: 霓虹绿双线（主线 4px + 辅助线 2px）
- **底部装饰线**: 霓虹绿双线（辅助线 4px + 主线 4px）
- **像素方块**: 角落使用递减透明度的方块装饰（100%→60%→30%）
- **扫描线网格**: 可选的低透明度背景网格线

## 六、页面类型

### 1. 封面页 (01_cover.svg)

- 深空黑背景
- 顶部/底部霓虹装饰线
- 像素风控制台图形（可选）
- 主标题（霓虹绿发光效果）
- 副标题（月光白）
- 功能按钮组（横向排列）
- 底部提示语（如 "PRESS START"）

### 2. 目录页 (02_toc.svg)

- 深空黑背景
- 标准顶部装饰
- 章节列表（带重要度标签）
  - 🔴 关键必学
  - 🟡 建议了解
  - 🟢 可选内容
- 像素风列表样式

### 3. 章节页 (02_chapter.svg)

- 深空黑背景
- 全屏霓虹效果
- 大号章节编号（发光效果）
- 章节标题 + 英文副标题
- 像素风装饰框

### 4. 内容页 (03_content.svg)

- 深空黑背景
- 标准顶部装饰
- 页面标题（霓虹绿 + 发光）
- 英文副标题（灰雾色）
- **内容区域完全开放**（y=140 到 y=670，宽度 1160px）
- 底部页码

> ⚠️ **设计原则**: 内容页模板仅提供页面框架（标题区 + 页脚），内容区域由 Executor 根据实际内容自由设计。可使用的布局包括但不限于：卡片、进度条、表格、时间线、对比图等。

### 5. 结束页 (04_ending.svg)

- 深空黑背景
- 霓虹发光主标题
- 总结卡片组
- "GAME SAVED" 视觉效果
- 进度按钮组

## 九、视觉特效

### 像素风格特征

- 方块化图标和装饰
- 使用 █ ▓ ░ ▀ ▄ ◼ ◻ 等字符装饰
- 进度条使用方块填充
- 边框使用双线或点状模拟
- 卡片角落添加像素装饰块

### 霓虹发光效果

对关键文字/元素添加发光滤镜：

```xml
<defs>
  <filter id="glowGreen" x="-50%" y="-50%" width="200%" height="200%">
    <feGaussianBlur stdDeviation="3-4" result="blur" />
    <feMerge>
      <feMergeNode in="blur" />
      <feMergeNode in="SourceGraphic" />
    </feMerge>
  </filter>
</defs>

<!-- 使用方式 -->
<text filter="url(#glowGreen)" fill="#39FF14">发光文字</text>
```

> ⚠️ **注意**: `filter` 效果在 PPT 中可能不完全兼容，但不会导致渲染失败。

## 十、SVG 技术约束

### 必须遵守

1. viewBox: `0 0 1280 720`
2. 背景使用 `<rect>` 元素
3. 文本换行使用 `<tspan>`（禁止 `<foreignObject>`）
4. 透明度使用 `fill-opacity` / `stroke-opacity`，禁止 `rgba()`
5. 禁止使用：`clipPath`、`mask`、`<style>`、`class`、`foreignObject`
6. 禁止使用：`textPath`、`animate*`、`script`、`marker`/`marker-end`
7. 箭头使用 `<polygon>` 三角形替代 `<marker>`

### PPT 兼容性规则

- 禁止 `<g opacity="...">`（组透明度），每个子元素单独设置
- 图片透明度使用遮罩层替代
- 仅使用内联样式，禁止外部 CSS 和 `@font-face`
- `filter` 效果作为增强（警告级别），不影响基础显示

## 十一、占位符规范

模板使用 `{{PLACEHOLDER}}` 格式的占位符，常用占位符：

| 占位符             | 说明               |
| ------------------ | ------------------ |
| `{{TITLE}}`        | 主标题             |
| `{{SUBTITLE}}`     | 副标题             |
| `{{AUTHOR}}`       | 作者/单位          |
| `{{PAGE_TITLE}}`   | 页面标题           |
| `{{PAGE_TITLE_EN}}`| 页面英文标题       |
| `{{CHAPTER_NUM}}`  | 章节编号           |
| `{{PAGE_NUM}}`     | 页码               |
| `{{TOTAL_PAGES}}`  | 总页数             |
| `{{VERSION}}`      | 版本号             |

---

## 十二、使用说明

1. 复制模板到项目 `templates/` 目录
2. 根据内容需求选择合适的页面模板
3. 使用占位符标记需要替换的内容
4. 通过 Executor 角色生成最终 SVG
5. 发光效果使用 `filter` 定义（在 `<defs>` 中）
6. 注意保持霓虹色系的一致性
