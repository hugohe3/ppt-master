# 政府红模板 (Government Red Style) - 设计规范

> 适用于各级政府机构政务汇报、政策宣讲、工作总结、项目推介等场景。

---

## 一、模板概述

| 属性           | 描述                                                         |
| -------------- | ------------------------------------------------------------ |
| **模板名称**   | government_red（政府红模板）                                 |
| **适用场景**   | 政务汇报、政策解读、工作总结、项目推介、招商引资             |
| **设计调性**   | 权威、庄重、专业、现代化政务风格                             |
| **主题模式**   | 亮色主题（白色背景 + 政府红/蓝强调色）                       |

## 五、页面结构

### 通用布局

| 区域       | 位置/高度     | 内容说明                               |
| ---------- | ------------- | -------------------------------------- |
| **顶部**   | y=0, h=6px    | 双色渐变横条（红+蓝），贯穿整页        |
| **标题栏** | y=30, h=50px  | 章节序号块 + 标题文字 + 右上角徽标     |
| **内容区** | y=100, h=560px | 主要内容区域                          |
| **页脚**   | y=680, h=40px | 页码、单位名称、底部装饰线             |

### 导航栏设计

- **顶部装饰线**: 双色渐变（`#8B0000` → `#003366`），高度 6px，贯穿整页
- **底部装饰线**: 政府红 (`#8B0000`)，高度 4px，y=716
- **标题栏** (y=30):
  - 章节序号块: 政府红正方形 (50×50px)，白色数字居中
  - 标题文字: 距序号块 20px，28px 字号，`#1A1A1A`
  - 右上角徽标: 固定于 x=1107，尺寸 113×50px

## 六、页面类型

### 1. 封面页 (01_cover.svg)

- 深色渐变背景（政府蓝为主）
- 顶部金色装饰线
- 主标题 + 副标题（居中白色）
- 单位名称：重庆两江新区发展和改革委员会
- 底部日期区域

### 2. 目录页 (02_toc.svg)

- 白色背景 + 左侧红色竖条装饰
- 最多支持5个章节
- 编号采用中国红方块 + 白色数字
- 右侧可选数据展示区

### 3. 章节页 (02_chapter.svg)

- 深蓝色渐变背景
- 大号章节编号（半透明装饰）
- 章节标题 + 英文副标题
- 几何装饰元素

### 4. 内容页 (03_content.svg)

- 白色背景
- 标准导航栏（红色序号块）
- 灵活内容区域
- 支持多种布局模式

### 5. 结束页 (04_ending.svg)

- 深蓝色背景
- 居中感谢语
- 单位全称
- 联系方式/地址信息

## 九、SVG 技术约束

### 必须遵守

1. viewBox: `0 0 1280 720`
2. 背景使用 `<rect>` 元素
3. 文本换行使用 `<tspan>`（禁止 `<foreignObject>`）
4. 透明度使用 `fill-opacity` / `stroke-opacity`，禁止 `rgba()`
5. 禁止使用：`clipPath`、`mask`、`<style>`、`class`、`foreignObject`
6. 禁止使用：`textPath`、`animate*`、`script`、`marker`/`marker-end`
7. 箭头使用 `<polygon>` 三角形替代 `<marker>`

### PPT 兼容性规则

- 禁止 `<g opacity="...">`（组透明度），每个子元素单独设置
- 图片透明度使用遮罩层替代
- 仅使用内联样式，禁止外部 CSS 和 `@font-face`

## 十、占位符规范

模板使用 `{{PLACEHOLDER}}` 格式的占位符，常用占位符：

| 占位符             | 说明               |
| ------------------ | ------------------ |
| `{{TITLE}}`        | 主标题             |
| `{{SUBTITLE}}`     | 副标题             |
| `{{AUTHOR}}`       | 单位名称（中文）   |
| `{{AUTHOR_EN}}`    | 单位名称（英文）   |
| `{{PAGE_TITLE}}`   | 页面标题           |
| `{{CHAPTER_NUM}}`  | 章节编号           |
| `{{PAGE_NUM}}`     | 页码               |
| `{{DATE}}`         | 日期               |
| `{{LOGO_HEADER}}`  | 页眉徽标文件名     |
| `{{COVER_BG_IMAGE}}`| 封面背景图文件名  |

## 十一、使用说明

1. 复制模板到项目目录
2. 替换 images 目录中的徽标文件（如有）
3. 根据内容需求选择合适的页面模板
4. 使用占位符标记需要替换的内容
5. 通过 Executor 角色生成最终 SVG

---

## 十二、设计亮点

- **双色渐变顶部装饰**: 红蓝渐变体现政务风格
- **金色点缀元素**: 增添庄重感
- **几何装饰图案**: 现代化政务风格
- **清晰的视觉层级**: 确保信息传达效率
