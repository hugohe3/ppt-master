# 招商银行模板 (CMB Bank Style) - 设计规范

> 适用于招商银行及金融机构的工作汇报、业务展示、年度报告、客户服务等场景。

---

## 一、模板概述

| 属性         | 描述                                                 |
| ------------ | ---------------------------------------------------- |
| **模板名称** | cmb_bank（招商银行模板）                             |
| **适用场景** | 工作汇报、业务展示、年度报告、产品推介、客户服务培训 |
| **设计调性** | 稳重、专业、可信赖、现代金融、服务至上               |
| **主题模式** | 亮色主题（白色背景 + 招行红标题条 + 金色点缀）       |
| **适用机构** | 招商银行各分支机构、金融服务企业                     |

---

## 二、画布规范

| 属性           | 值                         |
| -------------- | -------------------------- |
| **格式**       | 标准 16:9                  |
| **尺寸**       | 1280 × 720 px              |
| **viewBox**    | `0 0 1280 720`             |
| **页面边距**   | 左右 40px，上 0px，下 35px |
| **内容安全区** | x: 40-1240, y: 70-665      |

## 五、页面结构

### 通用布局

| 区域       | 位置/高度      | 内容说明                           |
| ---------- | -------------- | ---------------------------------- |
| **页眉**   | y=0, h=70px    | 招行红背景 + 金色装饰线 + 页面标题 |
| **观点条** | y=70, h=50px   | 核心观点/摘要区域（暖灰背景）      |
| **内容区** | y=135, h=515px | 主要内容区域                       |
| **页脚**   | y=665, h=55px  | 数据来源、机构名称、页码           |

### 装饰设计

- **金色装饰线**: 辅助金 (`#C9A962`)，用于页眉底部、分隔线
- **红色渐变**: 从招行红到深红的渐变效果
- **向日葵元素**: 可选的品牌装饰图案
- **圆角矩形**: 卡片使用 8px 圆角，体现现代感

## 六、页面类型

### 1. 封面页 (01_cover.svg)

- 白色背景
- 招行红顶部横条 + 金色底部装饰线
- 右上角 Logo/品牌标识占位区域
- 居中主标题 + 副标题
- 金色装饰分隔线
- 汇报人信息区域
- 底部暖灰信息区（日期）

### 2. 目录页 (02_toc.svg)

- 白色背景
- 标准页眉（招行红 + 金色线）
- 卡片式目录项布局（2列）
- 暖灰/浅红背景卡片 + 左侧红色竖条
- 可选项使用虚线边框

### 3. 章节页 (02_chapter.svg)

- 深红色全屏背景 (`#9A0E26`)
- 右侧几何图形装饰（金融主题）
- 左侧金色竖条装饰
- 大号半透明背景编号
- 醒目白色章节标题
- 金色装饰线

### 4. 内容页 (03_content.svg)

- 白色背景
- 标准页眉（招行红 + 金色线）
- 核心观点条（暖灰背景 + 红色左侧竖条）
- 灵活内容区域
- 页脚：数据来源、机构名称、页码

### 5. 结束页 (04_ending.svg)

- 白色背景
- 招行红顶部横条 + 金色装饰
- 居中感谢语
- 联系信息区域
- 品牌 Logo 区域

## 九、SVG 技术约束

### 必须遵守

- viewBox 固定为 `0 0 1280 720`
- 背景使用 `<rect>` 元素
- 使用 `<tspan>` 进行文本换行
- 所有颜色使用 HEX 格式（不使用 rgba）
- 透明度使用 `fill-opacity` / `stroke-opacity`

### 禁用元素（PPT 不兼容）

| 禁用项             | 替代方案               |
| ------------------ | ---------------------- |
| `clipPath`         | 不使用裁剪             |
| `mask`             | 不使用遮罩             |
| `<style>`          | 使用内联样式           |
| `class` / `id`     | 使用内联属性           |
| `foreignObject`    | 使用 `<tspan>` 换行    |
| `textPath`         | 使用普通 `<text>`      |
| `animate*` / `set` | 不使用动画             |
| `marker-end`       | 使用 `<polygon>` 箭头  |
| `<g opacity>`      | 每个元素单独设置透明度 |

## 十一、使用说明

### 1. 复制模板到项目

```bash
cp templates/layouts/cmb_bank/* projects/<项目>/templates/
```

### 2. Logo 放置建议

- 封面页：右上角，尺寸约 160×50px
- 内容页：右上角页眉内，尺寸约 120×35px
- 结束页：可放大展示，居中或配合联系信息

```xml
<rect x="x" y="y" width="200" height="110" fill="#FDF2F4" rx="8"/>
<rect x="x" y="y" width="4" height="110" fill="#C41230" rx="2"/>
<text x="x+100" y="y+35" text-anchor="middle" fill="#666666" font-size="14">存款余额</text>

<text x="x+100" y="y+98" text-anchor="middle" fill="#28A745" font-size="12">↑ 12.5%</text>
```

### 增长指标

```xml
<text fill="#28A745" font-size="14">↑ 15.2%</text>  <!-- 正增长 -->
<text fill="#DC3545" font-size="14">↓ 3.8%</text>   <!-- 负增长 -->
```
