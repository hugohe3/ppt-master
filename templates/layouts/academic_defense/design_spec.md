# 学术答辩风格 (Academic Defense) - 设计规范

> 适用于学术论文答辩、研究汇报、毕业设计展示等场景。

---

## 一、风格概述

| 属性               | 描述                                       |
| ------------------ | ------------------------------------------ |
| **风格名称** | academic_defense（学术答辩风格）           |
| **适用场景** | 毕业答辩、学术汇报、研究进展报告、课题申请 |
| **设计调性** | 专业、严谨、科研感、清晰层次               |
| **主题模式** | 亮色主题（白色背景 + 深蓝标题条）          |

## 二、画布规范

| 属性                 | 值                                       |

## 五、页面结构

### 通用结构

```
┌─────────────────────────────────────────────────────────┐
│ ▌章节编号 + 页面标题                    [Logo区域]     │  ← 页眉 (0-60)
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │ 核心观点/摘要区域（可选）                        │   │  ← 观点条 (60-115)
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐  │
│  │              │  │              │  │              │  │
│  │   内容卡片   │  │   内容卡片   │  │   内容卡片   │  │  ← 内容区 (115-670)
│  │              │  │              │  │              │  │
│  └──────────────┘  └──────────────┘  └──────────────┘  │
│                                                         │
├─────────────────────────────────────────────────────────┤
│ [页脚信息]                              [页码/导航]     │  ← 页脚 (670-720)
└─────────────────────────────────────────────────────────┘
```

### 页眉设计

- **背景**：深蓝色 `#003366` 横条，高度 60px
- **章节编号**：白色粗体，格式如 `1.1`、`3.2`
- **页面标题**：白色粗体，与编号间隔 1em
- **右上角**：品牌 Logo（像素答辩图标 + 文字）
- **装饰**：红色竖条 4px 宽，位于最左侧

### 核心观点条（可选）

- **背景**：浅蓝灰 `#E8F4FC`
- **高度**：55px
- **内边距**：左右 40px
- **文字**：正文黑 `#333333`，重点词用红色 `#CC0000` 加下划线

### 内容卡片

- **边框**：辅助蓝 `#0066CC`，1px 实线
- **圆角**：8px
- **内边距**：20px
- **标题区**：
  - 蓝底白字标签 `#0066CC`，圆角 4px
  - 或左侧 4px 蓝色竖条 + 深蓝标题文字
- **内容区**：白色或浅灰 `#F5F7FA` 背景

## 七、图标使用规范

### 推荐图标颜色

- **主要图标**：深蓝 `#003366` 或辅助蓝 `#0066CC`
- **强调图标**：强调红 `#CC0000`
- **中性图标**：正文灰 `#666666`

### 图标尺寸

- **大图标**：48 × 48 px（封面、章节页）
- **中图标**：32 × 32 px（卡片标题旁）
- **小图标**：20 × 20 px（列表项、标注）

## 八、技术约束

### SVG 生成规则

1. **viewBox**: `0 0 1280 720`
2. **背景**：使用 `<rect>` 元素填充
3. **文本换行**：使用 `<tspan>` 实现，禁止 `<foreignObject>`
4. **透明度**：使用 `fill-opacity` / `stroke-opacity`，禁止 `rgba()`
5. **禁止元素**：`clipPath`、`mask`、`filter`、`<style>`、`class`、`foreignObject`、`textPath`、`animate*`、`script`、`marker`

### PPT 兼容性

- 禁止 `<g opacity="...">`（组透明度）
- 图片透明度使用遮罩层
- 仅使用内联样式
