# 页面布局模板库

预置的页面布局模板，按设计风格分类。

## 目录结构

```
templates/layouts/
├── general/           # 通用灵活风格
├── consultant/        # 咨询风格
├── consultant_top/    # 顶级咨询风格（MBB级）
└── academic_defense/  # 学术答辩风格
```

## 4 个核心模板 + 1 个可选模板

每种风格包含 4 个核心模板，保持精简同时覆盖必要的结构性页面：

| 序号 | 文件名           | 用途   | 说明                                               |
| ---- | ---------------- | ------ | -------------------------------------------------- |
| 01   | `01_cover.svg`   | 封面   | 固定结构：标题、副标题、日期、机构                 |
| 02   | `02_chapter.svg` | 章节页 | 固定结构：章节编号、章节标题                       |
| 03   | `03_content.svg` | 内容页 | **灵活结构**：只定义页眉页脚，内容区由 AI 自由布局 |
| 04   | `04_ending.svg`  | 结束页 | 固定结构：感谢语、联系方式                         |
| --   | `02_toc.svg`     | 目录页 | **可选**：目录标题、章节列表                       |

> **设计理念**：模板定义视觉一致性和结构性页面，内容页保持最大灵活性，让 AI 根据实际内容决定布局。

## 风格特点

| 风格                 | 配色                       | 特点                         |
| -------------------- | -------------------------- | ---------------------------- |
| **general**          | 深蓝渐变 + 靛蓝紫          | 现代简洁，适合商业演示       |
| **consultant**       | 深蓝 + 金色线 + 左侧装饰条 | 专业严谨，咨询报告风格       |
| **consultant_top**   | 渐变顶条 + 网格 + 金色点缀 | MBB级，核心观点区 + 机密标识 |
| **academic_defense** | 深蓝 + 红色装饰 + 清晰层次 | 专业学术，适合毕业答辩/汇报  |

## 占位符规范

模板中使用以下占位符标记可替换内容：

| 占位符              | 用途        | 适用模板           |
| ------------------- | ----------- | ------------------ |
| `{{TITLE}}`         | 主标题      | 封面               |
| `{{SUBTITLE}}`      | 副标题      | 封面               |
| `{{DATE}}`          | 日期        | 封面               |
| `{{AUTHOR}}`        | 作者/机构   | 封面               |
| `{{CHAPTER_NUM}}`   | 章节编号    | 章节页             |
| `{{CHAPTER_TITLE}}` | 章节标题    | 章节页             |
| `{{CHAPTER_DESC}}`  | 章节描述    | 章节页             |
| `{{PAGE_TITLE}}`    | 页面标题    | 内容页             |
| `{{KEY_MESSAGE}}`   | 核心观点    | 内容页（咨询风格） |
| `{{CONTENT_AREA}}`  | 内容区域    | 内容页             |
| `{{SECTION_NAME}}`  | 章节名称    | 内容页页脚         |
| `{{SOURCE}}`        | 数据来源    | 内容页页脚         |
| `{{PAGE_NUM}}`      | 页码        | 内容页、结束页     |
| `{{THANK_YOU}}`     | 感谢语      | 结束页             |
| `{{TAGLINE}}`       | 口号/副标题 | 结束页             |
| `{{CONTACT_INFO}}`  | 联系信息    | 结束页             |
| `{{COPYRIGHT}}`     | 版权信息    | 结束页             |

## 使用方式

用户选择「A) 使用已有模板」时，可从以下来源获取模板：

1. **用户自备模板**: 已有的模板文件
2. **其他项目模板**: 从现有项目复制
3. **外部来源模板**: 网络下载、设计师提供等
4. **本示例库**: 从此目录复制（如下所示）

**从示例库复制**：

```bash
# 复制通用风格模板到项目
cp templates/layouts/general/* projects/<项目>/templates/

# 复制咨询风格模板到项目
cp templates/layouts/consultant/* projects/<项目>/templates/
```

复制后根据项目《设计规范》调整配色和文字。
