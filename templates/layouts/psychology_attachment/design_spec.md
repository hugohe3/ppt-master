# 心理疗愈模板 (Psychology Attachment Style) - 设计规范

> 适用于心理学、心理治疗、咨询培训、学术分享等专业场景。

---

## 一、模板概述

| 属性         | 描述                                             |
| ------------ | ------------------------------------------------ |
| **模板名称** | psychology_attachment（心理疗愈模板）            |
| **适用场景** | 心理治疗培训、学术讲座、咨询案例分析、专业分享   |
| **设计调性** | 专业、温暖、疗愈、信任感                         |
| **主题模式** | 浅色主题（云白背景 + 蓝绿渐变强调 + 多彩语义色） |

### 核心视觉隐喻

采用"**安全基地**"作为核心视觉隐喻：

- **结构稳固**：页面布局如同安全的依恋关系，有清晰的边界和可预期的模式
- **层次分明**：信息层级如同依恋系统的组织——从生物本能到高阶反思
- **温暖专业**：色彩既体现专业权威，又传递治愈温暖

## 六、页面类型

### 1. 封面页 (01_cover.svg)

- **背景**: 蓝绿渐变 (`#1E3A5F` → `#2E5C8E` → `#3D8B7A`)
- **装饰**: 可选背景图片（opacity=0.25）
- **标题区**: 居中，主标题 52px + 副标题 28px
- **英文标题**: 浅灰色，24px
- **装饰线**: 温暖橙色细线，200px 宽
- **底部**: 引言卡片（半透明背景 + 疗愈绿左边框）
- **标签**: 关键词标签（半透明胶囊）
- **页码**: 右下角，14px

### 2. 目录页 (02_toc.svg)

- **背景**: 云白 (`#F8FAFC`)
- **左装饰**: 主导色 8px 竖条
- **标题**: "内容概览" + "Contents Overview"
- **左侧**: 五章节列表（彩色编号方块 + 标题 + 描述）
  - 章节1: 主导蓝 `#2E5C8E`
  - 章节2: 疗愈绿 `#3D8B7A`
  - 章节3: 温暖橙 `#E07843`
  - 章节4: 冷漠灰蓝 `#64748B`
  - 章节5: 创伤红 `#B54545`
- **右侧**: 学习目标卡片
- **中间**: 虚线分隔

### 3. 章节页 (02_chapter.svg)

- **背景**: 蓝绿渐变
- **装饰**: 多个低透明度同心圆、斜线装饰
- **大号编号**: 120px，半透明白色居中
- **章节标签**: 胶囊形状 "CHAPTER X"
- **标题**: 48px 白色粗体
- **副标题**: 24px 浅灰英文
- **装饰线**: 温暖橙色细线，200px
- **引言**: 半透明引言卡片
- **关键词**: 底部标签组
- **页码**: 右下角

### 4. 内容页 (03_content.svg)

- **背景**: 云白
- **左装饰**: 主导蓝 8px 竖条
- **标题区**: 主标题 28px + 英文副标题 16px
- **分隔线**: 标题下方装饰线
- **内容区**: 灵活布局（三栏/左右分栏/单栏）
- **卡片样式**:
  - 白色背景 + 浅灰边框
  - 圆角 12-16px
  - 彩色顶部条 / 彩色左边框
- **底部提示**: 浅灰背景提示条（可选）
- **页码**: 右下角

### 5. 结束页 (04_ending.svg)

- **背景**: 蓝绿渐变
- **装饰**: 网络连接图（圆点 + 连线）
- **标题**: 主标题 56px + 副标题 28px
- **英文**: 浅灰英文标题
- **装饰线**: 温暖橙色细线，300px
- **信息区**: 半透明信息卡片
- **底部**: 版权信息

## 七、布局模式

### 7.1 三栏并列（对比/发现）

```
[卡片1: 360px] [间距: 40px] [卡片2: 360px] [间距: 40px] [卡片3: 360px]
```

- 每卡片: 彩色顶部条 + 图标 + 编号 + 标题 + 内容 + 底部标签
- 适用于: 三个发现、三种类型对比

### 7.2 左右分栏

```
[左栏: 560px] [间距: 60px] [右栏: 580px]
```

- 左侧: 概念/理论
- 右侧: 应用/实践
- 适用于: 概念解释、治疗关系

### 7.3 垂直堆叠（层级结构）

```
┌─────────────────────────────────┐
│       顶层：元认知               │
├─────────────────────────────────┤
│       表征层                     │
├─────────────────────────────────┤
│       情感层                     │
├─────────────────────────────────┤
│       躯体层                     │

```

- 适用于: 自我发展层级、理论框架

## 十、SVG 技术约束

### viewBox 规范

```xml
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 1280 720">
```

### 禁用功能（黑名单）

| 分类          | 禁用项                          |
| ------------- | ------------------------------- |
| **裁剪/遮罩** | ❌ `clipPath`, `mask`           |
| **样式系统**  | ❌ `<style>`, `class`, `id`     |
| **结构/嵌套** | ❌ `<foreignObject>`            |
| **文本/字体** | ❌ `textPath`, `@font-face`     |
| **动画/交互** | ❌ `<animate*>`, `<set>`, `on*` |
| **标记/箭头** | ❌ `marker`, `marker-end`       |

### PPT 兼容性规则

| ❌ 禁止                        | ✅ 正确替代                                   |
| ------------------------------ | --------------------------------------------- |
| `fill="rgba(255,255,255,0.1)"` | `fill="#FFFFFF" fill-opacity="0.1"`           |
| `stroke="rgba(0,0,0,0.5)"`     | `stroke="#000000" stroke-opacity="0.5"`       |
| `<g opacity="0.2">...</g>`     | 每个子元素单独设置 `opacity` / `fill-opacity` |

## 十二、使用说明

### 模板使用步骤

1. **复制模板**: 将模板文件复制到项目 `templates/` 目录
2. **替换占位符**: 使用实际内容替换 `{{}}` 占位符
3. **调整配色**: 根据主题微调配色方案
4. **生成内容**: 使用 Executor 角色生成具体页面
5. **后处理**: 运行 `finalize_svg.py` 完成图片嵌入

### 适用主题

- 心理治疗与咨询
- 依恋理论研究
- 发展心理学
- 临床案例分析
