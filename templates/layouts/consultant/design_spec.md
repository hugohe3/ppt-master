# 咨询顾问模板 (Consultant Style) - 设计规范

> 适用于商业咨询、战略分析、项目汇报、企业培训等场景。

---

## 一、模板概述

| 属性           | 描述                                                   |
| -------------- | ------------------------------------------------------ |
| **模板名称**   | consultant（咨询顾问模板）                             |
| **适用场景**   | 商业咨询、战略分析、项目汇报、企业培训                 |
| **设计调性**   | 专业、简约、商务、现代                                 |
| **主题模式**   | 深色主题（深蓝背景 + 蓝色强调色 + 金色点缀）           |

---

## 二、画布规范

| 属性           | 值                            |
| -------------- | ----------------------------- |
| **格式**       | 标准 16:9                     |
| **尺寸**       | 1280 × 720 px                 |
| **viewBox**    | `0 0 1280 720`                |
| **页面边距**   | 左右 60px，上 40px，下 40px   |
| **内容安全区** | x: 60-1220, y: 80-660         |

## 五、页面结构

### 通用布局

| 区域       | 位置/高度     | 内容说明                               |
| ---------- | ------------- | -------------------------------------- |
| **左侧**   | x=0, w=8px    | 蓝色装饰竖条                           |
| **页眉**   | y=0, h=60px   | 页面标题、Logo                         |
| **内容区** | y=80, h=540px | 主要内容区域                           |
| **页脚**   | y=660, h=60px | 数据来源、页码                         |

### 装饰设计

- **左侧装饰条**: 蓝色 (`#0066CC`)，宽度 8px，贯穿整页
- **金色分隔线**: 金色 (`#D4AF37`)，宽度 2-3px，用于标题装饰
- **几何装饰**: 透明边框矩形，营造现代感

## 六、页面类型

### 1. 封面页 (01_cover.svg)

- 深色背景 (`#0A1628`)
- 左侧蓝色装饰条
- 金色装饰线
- 主标题 + 副标题
- 右侧几何装饰
- 底部日期、作者信息

### 2. 目录页 (02_toc.svg)

- 深色背景
- 双竖线分隔符 `||` 设计
- 最多支持5个章节
- 左侧装饰竖线
- 右侧几何装饰

### 3. 章节页 (02_chapter.svg)

- 深色背景
- 蓝色章节编号块
- 章节标题 + 金色分隔线
- 右侧几何装饰

### 4. 内容页 (03_content.svg)

- 白色背景
- 深色页眉条
- 核心观点区（灰色背景 + 金色左侧装饰）
- 灵活内容区域
- 页脚包含数据来源和页码

### 5. 结束页 (04_ending.svg)

- 深色背景
- 居中感谢语
- 金色分隔线
- 联系信息
- 机密标识

## 九、SVG 技术约束

### 必须遵守

1. viewBox: `0 0 1280 720`
2. 背景使用 `<rect>` 元素
3. 文本换行使用 `<tspan>`（禁止 `<foreignObject>`）
4. 透明度使用 `fill-opacity` / `stroke-opacity`，禁止 `rgba()`
5. 禁止使用：`clipPath`、`mask`、`<style>`、`class`、`foreignObject`
6. 禁止使用：`textPath`、`animate*`、`script`、`marker`/`marker-end`
7. 箭头使用 `<polygon>` 三角形替代 `<marker>`

### PPT 兼容性规则

- 禁止 `<g opacity="...">`（组透明度），每个子元素单独设置
- 图片透明度使用遮罩层替代
- 仅使用内联样式，禁止外部 CSS 和 `@font-face`

## 十、占位符规范

模板使用 `{{PLACEHOLDER}}` 格式的占位符，常用占位符：

| 占位符             | 说明               |
| ------------------ | ------------------ |
| `{{TITLE}}`        | 主标题             |
| `{{SUBTITLE}}`     | 副标题             |
| `{{AUTHOR}}`       | 作者               |
| `{{DATE}}`         | 日期               |
| `{{PAGE_TITLE}}`   | 页面标题           |
| `{{CHAPTER_NUM}}`  | 章节编号           |
| `{{CHAPTER_TITLE}}`| 章节标题           |
| `{{KEY_MESSAGE}}`  | 核心观点           |
| `{{PAGE_NUM}}`     | 页码               |
| `{{SOURCE}}`       | 数据来源           |
| `{{CONFIDENTIAL}}` | 机密标识           |
| `{{LOGO}}`         | Logo文字           |

---

## 十一、使用说明

1. 复制模板到项目目录
2. 根据内容需求选择合适的页面模板
3. 使用占位符标记需要替换的内容
4. 通过 Executor 角色生成最终 SVG
