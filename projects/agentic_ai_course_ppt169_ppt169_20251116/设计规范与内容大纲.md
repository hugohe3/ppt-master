## 🎯 设计要点

### 视觉一致性
1. 所有卡片使用统一的圆角（12px）和阴影
2. 图标尺寸保持一致（标题旁 32px，卡片内 40px）
3. 时间戳格式统一：[HH:MM:SS]，使用 Gray 500 色

### 信息层级
1. 使用字重区分：Bold（标题）→ SemiBold（子标题）→ Regular（正文）
2. 使用颜色区分：主色（标题）→ 辅助色（重点）→ 灰色（说明）
3. 使用尺寸区分：48px（页面标题）→ 28px（卡片标题）→ 24px（正文）

### 技术规范
1. 所有 SVG 使用 `viewBox="0 0 1280 720"`
2. 背景使用 `<rect>` 元素
3. 文本换行使用 `<tspan>`，禁止 `<foreignObject>`
4. 使用系统字体栈，无需嵌入字体

### 数据可视化
1. 流程图使用圆角矩形 + 箭头
2. 矩阵图使用网格 + 象限填充
3. 光谱图使用渐变箭头
4. 协作图使用节点 + 连接线
