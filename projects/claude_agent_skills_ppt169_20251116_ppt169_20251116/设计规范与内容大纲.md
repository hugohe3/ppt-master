## 内容大纲

### 第 1 页：封面
- **标题**: Claude Agent Skills 完全指南
- **副标题**: 从入门到精通：打造可复用的 AI 工作流
- **视觉**: 大型图标 🤖 + 渐变背景
- **底部**: 日期与来源

### 第 2 页：什么是 Claude Skills？
- **标题**: 🤖 什么是 Claude Agent Skills？
- **核心概念**: 
  - Skills 是"可复用指令菜单"
  - 教 Claude 如何完成任务、使用什么工具、遵循什么标准
- **问题背景**: 
  - 过去：每次对话重复相同指令
  - Custom Instructions 的局限（不可移植、不可堆叠）
- **解决方案**: 账户级打包，跨平台可用

### 第 3 页：Skills 解决的两大痛点
- **标题**: Skills 解决的核心问题
- **痛点 1**: ❌ 不可移植 (Not Portable)
  - 指令锁定在特定项目中
- **痛点 2**: ❌ 不可堆叠 (Not Stackable)
  - 不同项目无法组合使用
- **解决方案**: ✅ 账户级可用 + ✅ 自由组合

### 第 4 页：Claude 生态系统全景
- **标题**: 🔑 Claude 生态系统：四大构建模块
- **四大模块**:
  1. 🧠 **Claude**: 核心大脑
  2. 📋 **Skills**: 指令菜单（如何做）
  3. 🔧 **MCP**: 工具平台（调用外部能力）
  4. 📁 **Projects**: 专属工作区（组合一切）
- **视觉**: 四象限或流程图

### 第 5 页：三种 Skills 类型
- **标题**: ✨ 三种 Skills 类型
- **类型 1**: 🏢 官方 Skills (Official Skills)
  - 由 Anthropic 构建
  - 在 Settings > Capabilities 启用
  - ⚠️ 只启用需要的，避免混淆
- **类型 2**: ⚙️ 自定义 Skills (Custom Skills)
  - 上传 skill 文件或使用 Skill Creator
- **类型 3**: 👥 社区 Skills (Community Skills)
  - ⚠️ 注意安全性

### 第 6 页：Pro Tips
- **标题**: 💡 使用技巧
- **Tip 1**: 在账户级自定义指令中添加提醒
  - "always consider relevant skills when responding"
- **Tip 2**: 选择性启用官方 Skills
- **Tip 3**: 安全使用社区 Skills

### 第 7 页：方法 1 - 扩展官方 Skills
- **标题**: 🛠️ 方法 1：扩展现有官方 Skills
- **示例**: 创建品牌化 PPT Skills
- **步骤**:
  1. 上传公司品牌模板
  2. 让 Claude 读取官方 PowerPoint Skill
  3. 创建扩展 Skill（提取颜色/字体）
  4. 上传生成的 zip 文件
- **结果**: 自动触发品牌化演示

### 第 8 页：方法 2 - 打包现有工作流
- **标题**: 🛠️ 方法 2：将现有工作流打包为 Skill
- **示例**: Notion 项目仪表盘
- **步骤**:
  1. 选择有详细指令的现有 Project
  2. 启用 Notion MCP
  3. 将全部指令打包为新 Skill
- **结果**: 在任何新对话中生成仪表盘

### 第 9 页：方法 3 - 从零构建多技能
- **标题**: 🛠️ 方法 3：从零构建多个 Skills
- **示例**: 博客内容创作工作流
- **步骤**:
  1. 创建 Skill 1: Keyword Research（调用 Ahrefs MCP）
  2. 创建 Skill 2: Blog Writer（AI 搜索优化）
  3. 在 Project 中组合使用
- **结果**: 一句话触发完整工作流
- **视觉**: 流程图展示

### 第 10 页：何时创建 Skill？三大判断标准
- **标题**: 🎯 何时创建 Skill？
- **三大判断标准**:
  1. ✅ 是否重复相同指令 3+ 次？
  2. ✅ 是否需要培训真人执行此流程？
  3. ✅ 是否需要每次保持质量/格式一致？
- **决策**: 满足 2+ 条 → 应创建 Skill
- **注意事项**:
  - Token 用量限制
  - Skills 维护（MCP 工具变更）
  - 添加备份方案

### 第 11 页：总结与行动建议
- **标题**: 💡 总结与下一步
- **核心要点**:
  - Skills = 可复用指令菜单
  - 三种类型 + 三种创建方法
  - 与 MCP、Projects 协同工作
- **立即行动**:
  1. 识别重复工作流
  2. 从方法 1 开始尝试
  3. 逐步构建个人 Skills 库
