# ChatGPT 进阶技巧 PPT - 设计规范与内容大纲

## 一、项目基本信息

- **项目名称**：ChatGPT 进阶技巧培训 PPT
- **画布格式**：PPT 16:9 (1280×720)
- **设计风格**：通用灵活风格
- **页数**：8 页
- **目标受众**：职场专业人士、内容创作者、营销人员

## 二、字体系统规范（严格执行）

### 字体栈（全局统一）

```
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
```

### 字号等级体系（固定值，严格遵守）

| 元素类型 | 字号 | font-weight | 用途 |
|---------|------|-------------|------|
| **主标题** | 56px | 700 (bold) | 封面主标题 |
| **页面标题** | 42px | 700 (bold) | 内页大标题 |
| **次级标题** | 32px | 600 (semi-bold) | 技巧名称、卡片标题 |
| **正文标题** | 24px | 600 (semi-bold) | 小节标题、步骤标题 |
| **正文内容** | 20px | 400 (regular) | 正文描述、列表内容 |
| **辅助文本** | 16px | 400 (regular) | 注释、页码、小标签 |
| **强调文本** | 20px | 700 (bold) | 关键词高亮 |

### 字体使用规则（不可违背）

1. **禁止使用以上未定义的字号**
2. **同类元素必须使用相同字号**（如所有技巧名称都是 32px）
3. **行高统一**：正文行高 = 字号 × 1.5
4. **字间距**：letter-spacing: 0（不额外调整）

## 三、配色方案

## 四、布局规范

### 画布与边距

- **viewBox**：`0 0 1280 720`
- **安全边距**：60px（上下左右）
- **内容区域**：1160×600px

### 卡片规范

- **单卡片高度**：240-280px
- **双卡片高度**：每个 120-140px，间距 30px
- **卡片圆角**：12px
- **卡片内边距**：30px
- **卡片阴影**：`filter: drop-shadow(0 4px 12px rgba(0,0,0,0.08))`

### 间距体系

- **大间距**：40px（主要区块之间）
- **中间距**：30px（卡片之间）
- **小间距**：20px（文本段落之间）
- **微间距**：12px（列表项之间）

## 六、设计元素规范

### 图标系统

- **样式**：线性图标，2px 线宽
- **尺寸**：48×48px（卡片图标）、32×32px（小图标）
- **颜色**：使用主题色或辅助色

### 列表样式

- **项目符号**：实心圆点或数字
- **缩进**：24px
- **行间距**：字号 × 1.5

### 强调样式

- **关键词**：bold + 主题色
- **引用框**：左侧 4px 竖线 + 浅色背景
- **代码/提示词**：等宽字体感觉的样式（使用背景色区分）
