# AI 数据分析框架 PPT 设计规范

## 项目信息
- **项目名称**：AI 数据分析 ACHIEVE & DIG 框架
- **画布格式**：PPT 16:9 (1280×720)
- **设计风格**：通用灵活风格
- **页数**：13 页
- **目标受众**：数据分析师、产品经理、技术团队

## 设计规范

### 画布与布局
- **viewBox**: `0 0 1280 720`
- **边距**：60px（上下左右）
- **安全区域**：60px - (1280-60) × 60px - (720-60)
- **内容区域**：1160 × 600

### 色彩方案
**主色调**：科技蓝 + 现代紫
- **主色**：#2563EB（蓝色，用于标题和强调）
- **辅助色 1**：#7C3AED（紫色，用于框架要点）
- **辅助色 2**：#10B981（绿色，用于成功/完成状态）
- **辅助色 3**：#F59E0B（橙色，用于警示/注意）
- **中性色**：#64748B（灰色，用于正文）
- **深色**：#1E293B（深灰，用于标题）
- **背景色**：#FFFFFF（白色）
- **卡片背景**：#F8FAFC（浅灰）

### 字体规范
使用系统 UI 字体栈：
```
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif
```

**字号层级**：
- **封面标题**：56px，font-weight: 700
- **封面副标题**：28px，font-weight: 400
- **页面标题**：42px，font-weight: 700
- **章节标题**：36px，font-weight: 600
- **卡片标题**：24px，font-weight: 600
- **正文大**：20px，font-weight: 400
- **正文中**：18px，font-weight: 400
- **正文小**：16px，font-weight: 400

### 卡片布局规范

#### 单列布局（全宽内容）
- **宽度**：1160px
- **高度**：根据内容调整（建议 120-200px）

#### 双列布局
- **卡片宽度**：550px
- **卡片间距**：60px
- **卡片高度**：根据内容调整（建议 240-280px）

#### 框架要点卡片（ACHIEVE 6个，DIG 3个）
- **宽度**：1160px
- **高度**：80-100px（单行要点）
- **间距**：20px
- **圆角**：12px
- **内边距**：24px

### 图标与装饰
- **emoji 使用**：保留原文档中的 emoji（🤖🔑✨🚀等）
- **编号样式**：圆形背景 + 白色数字
- **分隔线**：1px，颜色 #E2E8F0

### 视觉层级
1. **对齐**：所有元素左对齐或居中对齐，保持统一网格
2. **对比**：通过颜色、字号、粗细建立清晰层级
3. **重复**：相同类型的卡片使用一致的样式
4. **亲密性**：相关内容紧密排列，不相关内容留白
