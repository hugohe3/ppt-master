# AIRI Lab Template

## I. Template Overview

AIRI Lab is a dark, image-led brand deck for architectural AI, design workflow, and built-environment innovation presentations. The package is created in mirror mode from the reference PPTX, so each SVG page preserves the source slide composition as a reusable roster item rather than a placeholder skeleton.

## II. Color Scheme

- Deep architectural base: `#015B57` and `#000000`
- Highlight accent: `#FFC000` / `#F2AD00`
- Text and fine lines: `#EFEFEF`, `#FFFFFF`, and muted gray
- Source theme facts: accent4 is `#FFC000`; theme fonts are Calibri Light and Calibri.

## III. Signature Design Elements

- Use dark full-slide backgrounds and restrained yellow accents to frame architectural imagery.
- Keep the AIRI Lab logo assets in the upper-left brand zone when using mirrored source pages.
- Prefer large image crops, full-bleed visuals, and sparse text blocks over dense paragraph pages.
- Keep Chinese body copy short and use conservative font sizes when adapting this visual language downstream.
- Treat mirror pages as visual references: they preserve source text and images, so new decks should adapt the style carefully instead of assuming every page has editable placeholders.

The `assets/` directory mirrors the imported source media so the copied SVG pages remain visually faithful.
