# PPT 生成工作流

> ⚠️ **执行前置要求**：先阅读 [AGENTS.md](../../AGENTS.md)。本文件只负责串联入口，不重复展开各阶段规则。
>
> 本工作流现在是包装入口，用于串联两个独立阶段：
>
> 1. [generate-outline.md](./generate-outline.md) — 先完成《设计规范与内容大纲》与图片素材准备
> 2. [render-from-outline.md](./render-from-outline.md) — 再基于已就绪的大纲和素材生成 SVG / 讲稿 / PPT

## 推荐用法

- 只做内容规划与素材准备：使用 [generate-outline.md](./generate-outline.md)
- 已有大纲和素材继续出 PPT：使用 [render-from-outline.md](./render-from-outline.md)
- 需要串联完整流程：继续使用本入口，并参考 [AGENTS.md](../../AGENTS.md)
