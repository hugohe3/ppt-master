## 二、设计规范

### 1. 画布设置

- **尺寸**: 1280×720px
- **viewBox**: `0 0 1280 720`
- **安全区域**: 1200×640px（40px边距）
- **背景**: 使用 `<rect>` 元素，深蓝色或白色

### 2. 颜色方案

**主色调（咨询风格深蓝系）**:
```
主色 - 深海蓝:    #003F6C  (BCG 风格，稳重权威)
辅色 - 智慧蓝:    #0076A8  (德勤风格，专业可靠)

警示色 - 科技橙:  #FF6B35  (创新/数字化主题)
```

**辅助色系（图表/数据可视化）**:
```
图表蓝1: #2196F3
图表蓝2: #64B5F6
图表绿:  #4CAF50

```

**中性色系**:
```

辅助文本:   #8888A0  (注释说明)

边框色:     #E8E8F0
```

### 3. 字体规范

**字体栈**:
```
system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Microsoft YaHei', 'PingFang SC', sans-serif
```

**字号层级（高端咨询风格）**:
```
H1 - 封面标题:     48-52px, Bold, #FFFFFF 或 #1A1A2E
H2 - 页面标题:     32-36px, Bold, #1A1A2E
H3 - 章节标题:     24-28px, SemiBold, #003F6C
正文 - 主要内容:   18-20px, Regular, #4A4A6A
小字 - 注释说明:   14-16px, Regular, #8888A0
数据 - KPI数字:    40-56px, Bold, #003F6C 或强调色
```

**行高**:
```
标题行高: 1.2-1.3
正文行高: 1.5-1.6
```

### 4. 布局规范

**标准布局区域**:
```
标题区:    x=40, y=30, 高度=80px
内容区:    x=40, y=120, 宽度=1200, 高度=520px
页脚区:    x=40, y=660, 高度=40px
```

**卡片规范**:
```
单行卡片: 高度 530-580px
双行卡片: 每行 250-280px，间距 30px
三栏卡片: 每栏宽度 370px，间距 40px
四栏卡片: 每栏宽度 270px，间距 30px
```

**圆角**:
```
大卡片: 12-16px
小卡片: 8-12px
按钮/标签: 6-8px
```

### 5. 技术约束

- ✅ 使用 `<tspan>` 进行手动换行，通过 `x` 和 `dy` 控制定位
- ❌ 禁止使用 `<foreignObject>`
- ✅ 背景使用 `<rect>` 元素
- ✅ 遵循 CRAP 设计原则（对齐、对比、重复、亲密性）
- ✅ 数据可视化优先（图表、KPI卡片、矩阵）
- ✅ 页码显示在右下角
