# 演示文稿设计规范与内容大纲

## 项目信息

- **项目名称**：重庆大学硕士论文撰写格式标准
- **来源文档**：《重庆大学博士、硕士学位论文撰写格式标准及要求（2019 年修订）》
- **目标受众**：即将撰写硕士/博士论文的研究生
- **使用场景**：个人学习参考、开题/答辩汇报、研究生培训
- **创建日期**：2025 年 12 月 1 日

## 4. 排版体系

### 字体组合

- **中文**：系统默认无衬线字体（Microsoft YaHei, PingFang SC, sans-serif）
- **英文/数字**：Arial, Helvetica, sans-serif

### 字号层级

| 层级    | 用途      | 字号 | 字重   |
| ------- | --------- | ---- | ------ |
| H1      | 封面标题  | 48px | bold   |
| H2      | 章节标题  | 36px | bold   |
| H3      | 卡片标题  | 24px | bold   |
| Body    | 正文内容  | 18px | normal |
| Small   | 辅助说明  | 14px | normal |
| Caption | 脚注/页码 | 12px | normal |

- **核心主题**：摘要写作要求与格式
- **布局方式**：左右对比布局
- **主要内容**：

### Slide 06 - 主体部分撰写规范

- **文件名**：`slide_06_main_body.svg`
- **核心主题**：引言、正文、结论的要求
- **布局方式**：三栏并列卡片
- **主要内容**：
  - **引言（绪论）**：
    - 研究目的、范围、前人工作
    - 可单独成章进行文献综述
  - **正文**：
    - 核心研究内容
    - 各章后需有"本章小结"
    - 硕士 ≥3 万字，博士 ≥5 万字
  - **结论**：
    - 总体结论，非小结重复
    - 不标注引用文献

- **文件名**：`slide_07_figures_tables.svg`
- **核心主题**：图、表、公式的格式要求
- **布局方式**：三栏对比展示

  - **图**：

  - **表**：

## 7. 设计注意事项

1. **信息层级清晰**：使用颜色和字号区分标题、正文、备注
2. **数据可视化**：对数字参数用表格或卡片形式呈现
3. **对比展示**：正确/错误示例用颜色区分
4. **关键信息高亮**：重要数字和规范用强调色标注
5. **留白适度**：保持页面呼吸感，避免信息过载
