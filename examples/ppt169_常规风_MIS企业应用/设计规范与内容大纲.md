### **Slide 03：什么是企业系统 (ERP)**

**内容要点：**

- 标题：企业系统 (Enterprise Systems)
- 核心定义：基于集成软件模块 + 通用中央数据库
- 运作方式图示：
  - 生产 → 中央数据库 ← 财务
  - 销售 → 中央数据库 ← 人力资源
- 关键特点：实时数据共享，流程自动触发
- 示例：销售订单 → 自动验证信用 → 安排发货 → 更新库存 → 更新财务

**视觉方案：**

- 中心放置"中央数据库"图标
- 四周环绕四个部门模块（生产、财务、销售、人力资源）
- 双向箭头连接
- 使用 ERP 深蓝色调

### **Slide 05：供应链概述**

**内容要点：**

- 标题：供应链 (Supply Chain)
- 定义：采购原材料 → 转化为产品 → 分销给客户 的组织网络
- 图示：
  - 上游 (Upstream)：供应商的供应商 → 供应商 → [公司]
  - 下游 (Downstream)：[公司] → 分销商 → 零售商 → 客户
- 关键问题：牛鞭效应 (Bullwhip Effect)
  - 零售端微小波动 → 供应链逐级放大 → 过量库存

**视觉方案：**

- 水平流程图展示供应链
- 上游用浅绿色，下游用深绿色
- 底部单独卡片解释牛鞭效应
- 使用波浪线图形象化需求放大

---

### **Slide 06：SCM 软件与推拉模型**

**内容要点：**

- 标题：供应链管理软件
- 两类 SCM 软件对比：
  - 计划系统：需求预测、采购计划、库存决策
  - 执行系统：物流跟踪、仓库管理、配送优化
- 从推动到拉动：
  - 推动模型 (Push)：预测驱动 → 为库存生产
  - 拉动模型 (Pull)：订单驱动 → 按需生产
  - 互联网使拉动模型成为可能

**视觉方案：**

- 上半部分：两列对比卡片（计划 vs 执行）
- 下半部分：推拉模型对比图
- 使用箭头方向表示推/拉
- SCM 绿色主调

### **Slide 07：客户关系管理 (CRM) 概述**

**内容要点：**

- 标题：客户关系管理系统

- CRM 定义：捕获 → 整合 → 分析 → 分发客户数据
- 客户接触点：电话、邮件、网站、零售店、社交媒体
- 核心问题：
  - 谁是最有价值的客户？
  - 他们想要什么？

**视觉方案：**

- 中心放置"客户"图标

### **Slide 09：运营型 CRM vs 分析型 CRM**

**内容要点：**

- 标题：两种 CRM 类型
- 对比表格/图示：
  - 运营型 CRM：
    - 面向客户的应用
    - SFA、呼叫中心、营销自动化
    - 产生客户数据
  - 分析型 CRM：
    - 分析客户数据
    - 提供业务洞察
    - 核心产出：客户终身价值 (CLTV)
- CLTV 公式说明：收入 - 成本 × 关系生命周期

**视觉方案：**

- 左右对比布局
- 中间箭头连接（数据流动）
- 底部突出 CLTV 概念

## 🔧 技术实施注意事项

1. **SVG 画布**：`width="1280" height="720" viewBox="0 0 1280 720"`
2. **背景**：使用 `<rect>` 元素定义，不使用 CSS
3. **文本换行**：使用 `<tspan dy="行高">` 手动换行，禁止 `<foreignObject>`
4. **字体**：统一使用系统 UI 字体栈
5. **颜色一致性**：严格遵循色彩系统定义
6. **布局对齐**：遵循 CRAP 原则，确保元素对齐
