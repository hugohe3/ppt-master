## 📁 项目结构

每个示例项目采用以下标准结构：

```
<project_name>/
├── 设计规范与内容大纲.md      # 设计规范文档（必需）
├── images/                   # 图片资源（如有）
├── svg_output/               # 原始 SVG（带占位符，作为模板）
│   ├── slide_01_xxx.svg
│   └── ...
└── svg_final/                # 最终 SVG（嵌入图标/图片后）
    ├── slide_01_xxx.svg
    └── ...
```

## 📖 使用说明

### 预览项目

**方法 1: 使用 HTTP 服务器（推荐）**

```bash
# 预览最终版本
python -m http.server --directory examples/<project_name>/svg_final 8000

# 预览原始版本（带占位符）
python -m http.server --directory examples/<project_name>/svg_output 8000

# 访问 http://localhost:8000
```

**方法 2: 直接打开 SVG**

```bash
# macOS
open examples/<project_name>/svg_final/slide_01_cover.svg

# Windows
start examples/<project_name>/svg_final/slide_01_cover.svg

# Linux
xdg-open examples/<project_name>/svg_final/slide_01_cover.svg
```
