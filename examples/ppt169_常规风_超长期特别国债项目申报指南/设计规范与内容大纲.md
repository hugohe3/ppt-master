## 一、整体设计规范

### 1.1 画布与布局参数

```
画布尺寸: 1280 × 720 px
viewBox: 0 0 1280 720
边距: 60px (上/下/左/右)
内容安全区: 1160 × 600 px
栏间距: 40px
卡片圆角: 12px
```

### 1.2 视觉主题

**主题**: 亮色主题（政务专业风）

**理由**: 政策解读类内容需要清晰易读，亮色背景配合专业蓝色调能够体现权威性和可信度，同时保持良好的阅读体验。

### 1.3 色彩方案

| 角色         | 色值               | 用途                           |
| ------------ | ------------------ | ------------------------------ |
| **主导色**   | `#1E5AAF` (政务蓝) | 标题、重点强调、图标           |
| **辅助色 1** | `#2E8B57` (通过绿) | 正向信息、"允许"标识、成功状态 |
| **辅助色 2** | `#DC3545` (警示红) | 禁止规定、警告提示、否决条件   |
| **辅助色 3** | `#F59E0B` (提醒橙) | 注意事项、特殊说明             |
| **背景色**   | `#F8FAFC` (浅灰白) | 页面底色                       |
| **卡片背景** | `#FFFFFF` (纯白)   | 内容卡片                       |
| **文字主色** | `#1F2937` (深灰)   | 正文文字                       |
| **文字次色** | `#6B7280` (中灰)   | 辅助说明                       |

### 1.4 排版体系

| 层级    | 字号 | 字重     | 用途               |
| ------- | ---- | -------- | ------------------ |
| H1      | 42px | Bold     | 封面主标题         |
| H2      | 32px | Bold     | 页面标题           |
| H3      | 24px | SemiBold | 章节标题、卡片标题 |
| H4      | 20px | Medium   | 子标题             |
| Body    | 18px | Regular  | 正文内容           |
| Caption | 14px | Regular  | 注释、来源说明     |

**字体栈**: `"Microsoft YaHei", "PingFang SC", "Helvetica Neue", Arial, sans-serif`

### 1.5 卡片规范

| 类型     | 尺寸            | 说明               |
| -------- | --------------- | ------------------ |
| 全宽卡片 | 1160 × 自适应   | 流程图、时间轴     |
| 双栏卡片 | 560 × 280~320px | 左右对比、并列内容 |
| 三栏卡片 | 360 × 240~280px | 三要素展示         |
| 四栏卡片 | 265 × 200~240px | 四要点列表         |

### 1.6 图标风格

- **风格**: 线性图标 + 圆形背景
- **图标大小**: 32-48px
- **背景圆**: 60-80px 直径
- **颜色**: 主导色或对应语义色

### Page 12 - 申报流程图解

**布局**: 上下双流程图

- 标题: "💻 线上申报"

## 五、执行者指引

### 5.1 页面生成顺序

建议按以下顺序生成，便于保持视觉一致性：

1. 封面 (Page 01) - 确立整体视觉风格
2. 目录 (Page 02) - 确立卡片样式
3. 章节封面 (Page 03, 08, 10) - 批量生成
4. 内容页 - 按章节顺序生成

### 5.2 注意事项

1. **文字换行**: 使用 `<tspan>` 实现，禁止使用 `<foreignObject>`
2. **背景**: 使用 `<rect>` 元素填充背景色
3. **图标**: 使用 SVG path 绘制简洁的线性图标
4. **对齐**: 严格遵循 60px 边距，元素对齐到虚拟网格
5. **色彩使用**: 禁止性内容使用红色系，通过性内容使用绿色系，普通内容使用蓝色系
