# 霍夫斯泰德文化维度 - 微信公众号配图

## 项目概述

本项目为微信公众号文章《看不见的变量：决定项目生死的 5 个非财务指标》生成配图，基于 Hofstede 文化维度框架，从投资人视角解读组织行为学。

## 设计风格

**科技 + 商务融合风格**

- 深蓝色渐变背景，营造专业感
- 代码风格元素（Kernel/UI 隐喻）
- 数据可视化与对比图形
- 每个维度使用专属配色

## 技术规范

- **头图格式**: 900×383 px (2.35:1)
- **正文配图格式**: 1080×1080 px (1:1)
- **字体**: 系统 UI 字体栈
- **文本换行**: 使用 `<tspan>`，禁止 `<foreignObject>`

## 核心内容

文章基于 Hofstede 文化维度框架，阐述了 5 个决定项目生死的"非财务指标"：

1. **权力距离** - 对权力分配不平等的接受程度
2. **不确定性规避** - 面对模糊情境时的威胁感
3. **长期/短期导向** - 对未来回报的关注度
4. **个人/集体主义** - "我"vs"我们"的价值取向
5. **阳刚/阴柔气质** - 竞争导向 vs 生活导向

额外概念：**文化松紧度** - 组织规范的严格程度
