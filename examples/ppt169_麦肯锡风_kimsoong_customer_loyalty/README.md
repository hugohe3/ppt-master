## 项目背景

Kimsoong 是一家韩国汽车制造商，在欧洲市场以低价高质量的定位建立了品牌声誉。然而，面对激烈竞争，公司发现需要从"获取新客户"转向"留住现有客户"战略。

### 核心挑战

- **低复购率**: 仅 15% 的客户会再次购买
- **服务满意度低**: 67% 的客户认为售后服务一般或较差
- **客户流失**: 52% 流失到竞争对手，26% 因服务失望而流失
- **数据缺失**: 问卷回收率仅 40%
