# Kimsoong Customer Loyalty Programme - 设计规范与内容大纲

## 项目元数据

- **项目名称**: Kimsoong Customer Loyalty Programme
- **日期**: 2025-11-09
- **画布格式**: PPT 16:9 (1280×720)
- **风格定位**: 高端咨询风格（麦肯锡）
- **目标受众**: 企业高管、客户服务部门主管
- **预计页数**: 8 页

## 设计风格：高端咨询

### 核心设计原则

1. **数据驱动**: 以关键数据与洞察为核心，强化论据支撑
2. **结构化思维**: MECE 原则，清晰的逻辑框架
3. **信息可视化**: 图表、矩阵、漏斗模型优先
4. **专业留白**: 充足的呼吸感，避免视觉拥挤
5. **麦肯锡蓝**: 使用经典的 McKinsey Blue (#005587) 作为主色调

### 视觉规范

**配色方案**

- 主色：McKinsey Blue (#005587)
- 辅助色：Deep Teal (#004D5C)
- 强调色：Amber (#F5A623) - 用于关键数据
- 警示色：Coral (#E74C3C) - 用于问题点
- 中性色：
  - 深灰 (#2C3E50) - 标题
  - 中灰 (#5D6D7E) - 正文
  - 浅灰 (#ECF0F1) - 背景、分隔
  - 白色 (#FFFFFF) - 背景

**字体系统**

- 标题：Arial Bold, 32-44px
- 副标题：Arial Semibold, 24-28px
- 正文：Arial Regular, 16-18px
- 数据标注：Arial Bold, 20-24px
- 图表标签：Arial Regular, 14-16px

**布局规范**

- 页面边距：60px
- 标题区域高度：80-100px
- 内容区域：从 y=120 开始
- 图表间距：40-60px
- 文字行距：1.6

**图表规范**

- 条形图：宽度 500-700px，高度 400-500px
- 饼图：直径 300-400px
- 数据卡片：150×120px（KPI 展示）
- 矩阵：2×2 或 3×3，单元格 240-280px

## 内容结构大纲

### Slide 1: Cover - 战略提案封面

**布局**: 经典咨询封面

- 主标题："Kimsoong Customer Loyalty Programme"
- 副标题："Strategic Framework for Customer Retention"
- 底部：日期、项目代码
- 设计元素：McKinsey Blue 色块 + 简洁线条

### Slide 2: Situation - 当前挑战

**布局**: 问题陈述 + 关键数据

- 标题："Critical Challenges in Customer Retention"
- 核心问题框（左侧）：
  - 低复购率（15%）
  - 高客户流失（52% 转投竞品）
  - 服务满意度低（67% Fair/Poor）
- 数据可视化（右侧）：
  - 漏斗图：客户流失路径
  - 关键数字用 Amber 色强调

### Slide 3: Customer Profile - 客户画像分析

**布局**: 数据仪表板

- 标题："Understanding Our Customer Base"
- 四象限布局：
  1. 年龄分布（饼图）：48% < 30 岁
  2. 收入结构（条形图）：82% 中等收入
  3. 职业分布（条形图）
  4. 关键洞察文本框
- 底部：数据来源注释

### Slide 4: Customer Priorities - 客户需求矩阵

**布局**: 优先级矩阵 + 差距分析

- 标题："Customer Priorities vs. Current Performance"
- 2×2 矩阵：
  - X 轴：重要性（Customer Priority）
  - Y 轴：满意度（Current Rating）
  - 气泡：Economy, Price, Reliability, After-sales Service
- 标注："After-sales Service" 在高重要性低满意度区（红色警示）

### Slide 5: Root Cause Analysis - 流失原因

**布局**: 瀑布图/帕累托图

- 标题："Why Customers Leave: Root Cause Analysis"
- 横向条形图（降序）：
  - 竞品转换：52%（最长条，红色）
  - 服务失望：26%（橙色）
  - 搬迁：8%
  - 不再驾驶：5%
  - 未知：9%
- 累计百分比线
- 洞察文本框："78% 可控流失因素"

### Slide 6: Strategic Objectives - 战略目标

**布局**: 目标架构图

- 标题："Four Pillars of Customer Loyalty Strategy"
- 中央：核心目标（圆形）
- 四周：四大支柱（矩形卡片）：
  1. Build long-term relationships → Increase profits
  2. Increase customer loyalty → Boost repeat rate
  3. Accurate buyer profiling → Data-driven decisions
  4. Staff engagement → Service excellence
- 使用 McKinsey Blue 渐变

### Slide 7: Recommended Initiatives - 方案推荐

**布局**: 解决方案矩阵（2×2 或 1×5 卡片）

- 标题："Five Key Initiatives for Loyalty Programme"
- 五个方案卡片（优先级排序）：

  1. **3-Year Free After-sales Service** ⭐
     - 影响：直击痛点（服务不满）
     - 成本：中-高
  2. **20% Loyalty Discount**
     - 影响：直接财务激励
     - 成本：中
  3. **Enhanced Trade-in Programme**
     - 影响：降低换购门槛
     - 成本：低-中
  4. **Customer Magazine**
     - 影响：情感连接
     - 成本：低
  5. **Questionnaire Incentive (Branded Pen)**
     - 影响：提升数据质量
     - 成本：低

- 每个卡片包含：图标、名称、关键指标、成本标签

### Slide 8: Implementation Roadmap - 实施路线图

**布局**: 时间轴 + 里程碑

- 标题："Phased Implementation & Success Metrics"
- 时间轴（水平）：Q1-Q4
  - Phase 1 (Q1): 快赢项目（Magazine, Questionnaire）
  - Phase 2 (Q2): 服务提升（Free After-sales）
  - Phase 3 (Q3): 财务激励（Discount, Trade-in）
  - Phase 4 (Q4): 评估与优化
- 底部：KPI 追踪卡片
  - 目标复购率：15% → 30%
  - 服务满意度：33% → 60%（Good 以上）
  - 问卷回收率：40% → 70%

## SVG 技术规范

### 画布设置

```svg
<svg xmlns="http://www.w3.org/2000/svg"
     viewBox="0 0 1280 720"
     width="1280"
     height="720">
```

### 文本处理

- 使用 `<text>` + `<tspan>` 手动换行
- 禁止 `<foreignObject>`
- 字体栈：`font-family="Arial, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif"`

### 背景

```svg
<rect width="1280" height="720" fill="#FFFFFF"/>
```

### 图表元素

- 条形图：`<rect>` + `<text>`
- 饼图：`<path>` (arc) + `<text>`
- 线条：`stroke="#005587" stroke-width="2"`
- 阴影：`filter` with `feGaussianBlur`（可选）

### 麦肯锡风格特色

- 大量留白（内容占比 < 65%）
- 网格对齐（40px 基准网格）
- 数据为王（图表占比 > 50%）
- 极简图标（几何形状）
- 专业配色（避免花哨渐变）
