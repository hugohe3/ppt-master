### 第9页：配套设施用地获取方式（一）

**布局**: 上下分区

**核心内容**:

**问题提出**:
配套设施指非项目主体部分产生的收益：服务区、加油站、停车场、广告牌、食堂、充电桩等

**招拍挂方式的问题**:
1. ❌ 项目公司未必能成为使用权人 → 推翻资金平衡机制
2. ❌ 特许经营招标无法代替招拍挂程序 → 时序错配

**解决方案一：协议出让**

| 法律依据 | 适用情形 |
|---------|---------|
| 《城市房地产管理法》 | 商旅娱住有条件可协议 |
| 《协议出让国有土地使用权规定》 | 同一地块仅一个意向用地者 |

**可协议出让的配套设施**:
- 加油（气）站
- 农贸市场
- 社区综合体
- 养老设施
- 保障性安居工程
- 文体设施
- 历史文化保护
- 科教创新设施

### 第11页：配套设施用地获取方式（三）

**布局**: 左右双栏

**核心内容**:

**集体经营性建设用地**

使用范畴（新版《土地管理法》）:
- 国土空间规划确定为工业、商业等经营性用途
- 已依法办理土地所有权登记

适用项目类型:
- 休闲农业、乡村旅游
- 医养结合、露营民宿
- 文化体验、创意办公
- 电子商务
- 农村能源、农产品冷链
- 初加工、物流仓储

使用方式:
- 自营
- 出租
- 入股
- 联营
- 收益共享

---

**土地流转**

核心概念:
- 三权分置：所有权、承包权、经营权
- 仅限于农村土地使用权（经营权）流转

三种方式:

| 方式 | 说明 |
|------|------|
| 土地转包 | 村集体内部，不改变承包关系 |
| 政府返租 | 乡镇/村集体支付租金返租后再承包 |
| 作价入股 | 土地承包经营权出资入股项目公司 |

关键点:
- 农户主体地位（不能代签）
- 市场化发展机制
- 保障农户收益

**产业用地政策**

供应方式:
- 长期租赁
- 先租后让
- 租让结合

作价出资入股:

**存量资产盘活**

三类资产:

| 类型 | 具体内容 |
|------|---------|
| 经营性/准经营性 | 交通、水利、清洁能源、保租房、市政、环保、产业园、物流、旅游、新基建 |
| 存量+改扩建 | 交通枢纽改造、退城进园、污水厂下沉、地铁上盖、公交用地综合开发 |

**应用**: 通过剥离或整合，识别可特许经营的项目包，运用存量盘活政策红利

### 第13页：自然资源组合供应政策

**布局**: 流程图+要点框

**核心内容**:

**政策依据**: 自然资发〔2023〕89号

**组合供应概念**:
- 特定国土空间范围内
- 同一使用权人需整体使用多门类自然资源资产
- 一并公告、签订合同

**特许经营中的问题**:
- 公益性用地：划拨 ✓
- 经营性配套用地：招拍挂 ❌

招拍挂的障碍:
1. 项目公司未必能成为使用权人
2. 特许经营招标 ≠ 招拍挂
3. 经营性用地在招标阶段仍是"毛地"
4. 触及土地一二级联动禁区

**组合供应实现路径**:

```
特许经营项目
    │
    ├─ 公益性用地 ──→ 划拨
    │
    └─ 配套经营性用地 ──→ 协议出让 / 带方案出让 / 综合评价
                             │
                             ▼
                    精准锁定特许经营者获取土地使用权
```

**注意**: TOT项目需符合国资进场交易和产权管理要求
