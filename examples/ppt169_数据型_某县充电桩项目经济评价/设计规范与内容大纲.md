## 设计规范

### 1. 视觉风格定位

**高端咨询风格 - 央企专业版**
- 体现集团的权威性与专业性
- 强化数据可视化和信息层级
- 简洁、稳重、专业

### 2. 配色方案

**主色调**：
- **深蓝色**：#003F6C（主标题、关键图形）- 央企稳重感
- **商务蓝**：#005587（辅助色、图表）
- **强调橙**：#FF6B35（关键数据、正向指标）

**辅助色**：
- 浅灰背景：#F5F7FA
- 中性灰：#7D8FA8（说明文字）
- 深灰：#2C3E50（正文）
- 成功绿：#27AE60（正向趋势）
- 警示红：#E74C3C（风险提示）

**图表配色**：
- 柱状图/饼图：#005587、#0076A8、#4A90E2、#78B9EB、#A8D5F2
- KPI卡片：白底 + 深蓝边框 + 橙色数字

### 3. 字体规范

- **标题字体**：48-54px, 粗体, #003F6C
- **副标题**：32-36px, 中粗, #005587
- **正文**：18-20px, 常规, #2C3E50
- **数据标注**：16-18px, 中粗, #FF6B35
- **说明文字**：14-16px, 常规, #7D8FA8

字体栈：
```
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Microsoft YaHei', sans-serif
```

### 4. 布局规范

**画布**：1280×720px (16:9)

**边距**：
- 页面边距：60px (左右上下)
- 标题区高度：100-120px
- 内容区：520-540px

**网格系统**：
- 双栏布局：左侧560px，间距60px，右侧560px
- 三栏布局：每栏360px，间距60px
- KPI卡片：单个240×160px，间距40px

**对齐基准**：
- 标题左对齐：x=60
- 内容左对齐：x=80
- 图表居中或左对齐

### 5. 设计元素

**标题栏**：
- 位置：y=40-60
- 左侧：页面标题 + 细线装饰
- 右侧：集团标识区（文字形式）

**页码**：
- 位置：右下角 (1200, 680)
- 格式：当前页/总页数
- 样式：14px, #7D8FA8

**装饰元素**：
- 最小化装饰，以内容为核心
- 使用细线分割（1-2px, #E0E6ED）
- 关键数据用色块或边框强调

### 6. 图表规范

**KPI卡片**：
- 尺寸：240×160px 或 280×180px
- 白底 + 2px深蓝边框
- 数字：36-42px, 橙色
- 标签：16px, 深灰
- 单位/说明：14px, 中灰

**柱状图**：
- 柱宽：40-60px
- 柱间距：30-40px
- 坐标轴：1px, #E0E6ED
- 数值标注：柱顶上方5px

**折线图**：
- 线宽：3px
- 数据点：圆形, r=5
- 网格线：虚线, #E0E6ED

**饼图/环形图**：
- 外径：180-220px
- 内径（环形）：外径×0.6
- 标签引线：1px

**表格**：
- 表头：深蓝背景 + 白字
- 行高：40-45px
- 边框：1px, #E0E6ED
- 斑马纹：#F8F9FB

## 设计要点

1. **专业性**：所有数据精确展示，单位统一，计算逻辑清晰
2. **可读性**：关键指标用KPI卡片突出，图表辅助理解
3. **层级感**：标题-副标题-正文-注释，层次分明
4. **视觉流**：从左到右，从上到下，符合阅读习惯
5. **留白**：充足的呼吸空间，避免拥挤
6. **一致性**：所有页面风格统一，图表样式一致
