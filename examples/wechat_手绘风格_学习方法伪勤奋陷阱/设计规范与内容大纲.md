## 手绘风格定位

### 核心特征

- **不规则手绘线条**：模拟手绘笔触，线条略显不完美
- **温暖亲和色彩**：柔和的色调，避免纯正色彩
- **涂鸦式图标**：简化的手绘插画元素
- **纸质纹理感**：温暖的米白色背景
- **随性排版**：略带倾斜、不完全对齐的元素

### 情感定位

- 轻松、友好、易接近
- 去严肃化，增强共鸣感
- 适合年轻受众的视觉语言
- 营造"朋友聊天"般的氛围

## 设计系统

### 画布规格

- **尺寸**：900×383 px
- **viewBox**：`"0 0 900 383"`
- **宽高比**：2.35:1
- **安全边距**：左右 40px，上下 30px

### 色彩方案（手绘风格）

**主色调**：

- **手绘红**：`#FF6B6B` - 更柔和的红色，用于强调陷阱
- **温暖橙**：`#FFA07A` - 亲和的橙色调
- **柔和蓝**：`#4ECDC4` - 清新的青蓝色
- **深灰蓝**：`#2C3E50` - 文字主色

**辅助色**：

- **米白背景**：`#FFF8F0` - 纸质感背景
- **浅灰**：`#D1D8E0` - 次要元素
- **淡黄**：`#FFE66D` - 高亮与装饰

**纹理与效果**：

- 背景使用米白色代替纯白
- 线条使用 `stroke-linecap="round"` 和 `stroke-linejoin="round"` 模拟手绘
- 添加轻微的不规则性（通过路径点的微调）

### 字体系统

**字体栈**（保持一致）：

```
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "Helvetica Neue",
             "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif
```

**字号规范**：

- **大标题**：48-56px（封面主标题）
- **中标题**：36-42px（各页主标题）
- **正文**：24-28px（描述文字）
- **小字**：18-20px（辅助信息）

**字重**：

- 标题：700 (Bold)
- 正文：400 (Regular)
- 强调文字可略带倾斜（italic）增加手绘感

### 手绘元素设计原则

#### 线条特征

- **stroke-width**：3-5px（比标准风格粗一些）
- **stroke-linecap**：`round`（圆润端点）
- **stroke-linejoin**：`round`（圆润连接）
- **线条略带不规则**：通过曲线路径模拟手抖效果

#### 图标风格

- 简化的涂鸦式图标
- 不追求完美的几何形状
- 使用手绘线条而非填充
- 添加装饰性的小元素（星星、点、短线）

#### 装饰元素

- 手绘风格的虚线、波浪线
- 涂鸦箭头、星星、感叹号
- 不规则的圆圈和方框
- 手写风格的强调线

### 布局原则

**横版头图布局** (900×383)：

- **核心内容区**：820×323 px
- **左右布局**：图标区（略带随性位置）+ 文字区
- **或居中布局**：元素略带倾斜与错位
- **打破完美对齐**：营造手绘随性感

## 技术规范

### SVG 基础结构

```xml
<svg xmlns="http://www.w3.org/2000/svg"
     viewBox="0 0 900 383"
     width="900"
     height="383">
  <!-- 米白色背景 -->
  <rect width="900" height="383" fill="#FFF8F0"/>

  <!-- 内容层 -->
  <!-- ... -->
</svg>
```

### 手绘线条实现技巧

#### 1. 圆润端点

```xml
<line stroke-linecap="round" stroke-linejoin="round"/>
```

#### 2. 不规则曲线（模拟手抖）

```xml
<!-- 标准圆：完美几何 -->
<circle cx="100" cy="100" r="50"/>

<!-- 手绘圆：使用 path 创建略不规则的圆 -->
<path d="M 150 100 Q 150 60, 120 55 Q 90 50, 65 70 Q 40 90, 50 120 Q 60 150, 90 155 Q 120 160, 145 135 Q 160 110, 150 100" />
```

#### 3. 涂鸦箭头

```xml
<path d="M 50 50 Q 80 40, 100 50 L 95 45 M 100 50 L 95 55"
      stroke="#FF6B6B"
      stroke-width="4"
      stroke-linecap="round"
      fill="none"/>
```

### 文本换行规则

- 使用 `<text>` + `<tspan>` 进行手动换行
- **禁止使用 `<foreignObject>`**
- 每行 `<tspan>` 设置 `x` 和 `dy` 属性
- 可适当使用 `font-style="italic"` 增加手写感

### 对齐与间距

- **打破完美对齐**：元素可略带随机偏移（±5px）
- 行高：字号的 1.4-1.5 倍
- 段落间距：20-30px
- 元素间距：30-40px

## 手绘风格 vs 标准风格对比

| 特征   | 标准风格     | 手绘风格       |
| ------ | ------------ | -------------- |
| 背景色 | 浅灰 #ECF0F1 | 米白 #FFF8F0   |
| 线条   | 规整、几何   | 圆润、略不规则 |
| 图标   | 现代线性     | 涂鸦式         |
| 色彩   | 饱和、专业   | 柔和、温暖     |
| 排版   | 精确对齐     | 略带随性       |
| 装饰   | 简约几何     | 丰富涂鸦       |
| 情感   | 专业、清晰   | 友好、亲和     |

## 备注

- 手绘风格适合营造轻松、易接近的氛围
- 保持内容与标准版一致，仅改变视觉表现
- 不规则性要适度，避免影响可读性
- 米白背景在手机屏幕上更温暖舒适
