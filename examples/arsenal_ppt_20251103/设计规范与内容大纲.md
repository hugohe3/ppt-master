## 二、布局网格系统

### 画布结构（1280×720px）
```
┌────────────────────────────────────────────────┐
│  [80px]  ← 左边距                              │
│  ┌──────────────────────────────────────┐      │
│  │                                      │  80px │
│  │      1120px 内容安全区               │  →    │
│  │      高度：560px（上下各80px边距）    │  右   │
│  │                                      │  边   │
│  └──────────────────────────────────────┘  距   │
│                            [80px] ← 下边距      │
└────────────────────────────────────────────────┘
```

### 关键度量
- **左右边距**：80px  
- **上下边距**：80px  
- **内容区宽度**：1120px（80 + 1120 + 80 = 1280）  
- **内容区高度**：560px（80 + 560 + 80 = 720）

### 卡片布局基线
- **单卡片宽度**：1120px（全宽）  
- **双列卡片宽度**：540px（间距40px：540 + 40 + 540 = 1120）  
- **三列卡片宽度**：350px（间距35px：350 + 35 + 350 + 35 + 350 = 1120）  
- **标准卡片高度**：200px, 250px, 300px（根据内容选择）

## 三、图标/图片位置标注规则

### 标注方式
所有需要放置图片/图标的位置，使用**浅灰色矩形 + 虚线边框 + 居中文字说明**：

```svg
<!-- 示例：图片占位符 -->
<rect x="100" y="100" width="300" height="200" 
      fill="#F5F5F5" 
      stroke="#CCCCCC" 
      stroke-width="2" 
      stroke-dasharray="8,4" 
      rx="12"/>
<text x="250" y="200" 
      font-family="'PingFang SC', 'Microsoft YaHei', sans-serif" 
      font-size="18" 
      fill="#999999" 
      text-anchor="middle">
  [图片：队徽]
</text>
```

- **图表/数据可视化**：`[图表：类型说明]`

## 六、实施注意事项

### 1. SVG 技术规范
- **禁止使用 `<foreignObject>`**，所有文本使用 `<text>` + `<tspan>` 手动换行
- 背景使用 `<rect>` 元素
- 所有坐标、尺寸精确到像素

### 2. 图标占位符标准
- 填充色：`#F5F5F5`
- 边框：`#CCCCCC`，宽度2px，虚线 `stroke-dasharray="8,4"`
- 圆角：12px（小图标）/ 16px（大图片）
- 说明文字：18px，灰色 `#999999`，居中

### 3. 文本换行规则
- 单行最大宽度建议：500-550px
- 使用 `<tspan>` 的 `x` 和 `dy` 属性控制换行
- 行高 = 字号 × 行高系数

### 4. 视觉一致性
- 所有卡片使用统一圆角（16px）
- 所有阴影参数一致
- 标题位置在每页保持对齐
- 页码统一放置在右下角（1200, 690）
