### 视觉元素

#### 像素风格特征

- 方块化图标和装饰

- 进度条使用方块填充
- 边框使用双线或点状模拟

#### 发光效果（霓虹感）

对关键文字添加 `filter` 微光效果：
```svg
<filter id="glow">
  <feGaussianBlur stdDeviation="2" result="blur"/>
  <feMerge>
    <feMergeNode in="blur"/>
    <feMergeNode in="SourceGraphic"/>
  </feMerge>
</filter>
```

## 内容大纲

### 第 1 页：封面

- **主标题**: 🎮 Introducing...Git!
- **副标题**: 版本控制入门完全指南
- **装饰**: 像素风游戏控制台图形、闪烁的存档点图标
- **底部**: PRESS START TO LEARN

### 第 2 页：学习导航

- **标题**: 📍 本章学习导航
- **内容**: 使用重要度标签区分内容
  - 🔴 **关键必学**: 什么是 Git、Git vs GitHub、Git 能做什么
  - 🟡 **建议了解**: 谁在使用 Git
  - 🟢 **可选内容**: Git 的历史
- **布局**: 列表式，带图标标记

### 第 3 页：什么是 Git

- **标题**: 🤔 什么是 Git？
- **核心定义**: Git = 版本控制系统 (VCS)
- **内容**:
  - 追踪和管理文件变更的软件
  - 世界上最流行的 VCS
  - 其他 VCS：Subversion, CVS, Mercurial（已被淘汰）
- **数据可视化**: 90%+ 开发者使用 Git 的进度条

### 第 4 页：版本控制能做什么

- **标题**: 💾 版本控制能做什么？
- **功能列表**（像素风卡片）:
  - ⏪ 回到过去的版本
  - 🔍 比较不同版本的差异
  - ↩️ 撤销错误的修改
  - 🤝 与他人分享和协作
- **布局**: 2x2 网格卡片

### 第 5 页：游戏存档比喻

- **标题**: 🎮 Git 就像游戏存档点
- **视觉**: 横向游戏进度条
  - 玩家角色 → BOSS 战 → 死亡 → 读档重来
- **说明**: 
  - 做了存档 = 可以随时回来
  - 搞砸了？没关系，读档就好
- **风格**: 高度像素化的游戏场景

### 第 6 页：论文版本地狱

- **标题**: 📝 没有 Git 的痛苦
- **视觉**: 文件夹结构图
  ```
  essay_v1.doc
  essay_v2.doc
  essay_v2_final.doc
  essay_v2_final_REAL.doc
  essay_PRINT_THIS_ONE.doc
  ```
- **对比**: Git 的解决方案 - 一个项目，无限历史
- **布局**: 左右对比

### 第 7 页：可视化工作流

- **标题**: 🔀 Git 工作流可视化
- **视觉**: 时间线形式展示
  - 初始化 → 添加内容 → 提交 → 修改 → 提交
  - 分支 → 合并
- **说明**: 每个「存档点」= 一次 commit
- **布局**: 横向时间线 + 分支示意

### 第 8 页：Git 简史

- **标题**: 📜 Git 的诞生
- **时间线**:
  - 2005 年 4 月：Linus Torvalds 创建
  - 起因：BitKeeper 收费了
  - 几天内完成基础功能
- **趣闻**: Git 名字的含义
  - "愚蠢的内容追踪器"
  - "全球信息追踪器"
  - 英式俚语：讨厌的人
- **布局**: 左侧时间线，右侧趣闻卡片

### 第 9 页：Git ≠ GitHub

- **标题**: ⚠️ Git ≠ GitHub
- **对比表格**:
  | Git | GitHub |
  |-----|--------|
  | 软件 | 网站服务 |
  | 本地运行 | 云端托管 |
  | 不需要网络 | 需要网络 |
  | 免费开源 | 免费+付费 |
- **视觉**: 本地电脑 ↔ 云端服务器
- **关键点**: GitHub 是 Git 的托管平台之一

### 第 10 页：总结

- **标题**: 🎯 关键要点回顾
- **三大要点**:
  1. **Git 是什么**: 最流行的版本控制系统
  2. **Git 能做什么**: 追踪变更、时光旅行、协作
  3. **Git vs GitHub**: 本地软件 vs 云端服务
- **结束语**: READY TO START YOUR GIT JOURNEY?
- **装饰**: GAME SAVED ✓
