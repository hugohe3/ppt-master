本项目是基于依恋理论在心理治疗中应用的专业培训 PPT，系统阐述了从 Bowlby、Ainsworth 到 Main、Fonagy 等学者的核心理论，以及临床实践中的应用策略。

**适用场景：**

### 第一部分：理论基础（Slides 3-7）
- 依恋理论的三个核心发现

- 元认知与心智化

### 第二部分：依恋模式分类（Slides 8-12）
- 四种依恋模式总览

### 第四部分：临床应用（Slides 17-20）

- 从依恋到改变：临床实践路径
