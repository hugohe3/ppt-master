## ⚡ 核心要点速查

### 双轨并行机制（重中之重）

| 轨道     | 系统               | 主管部门    | 核心目标       |
| -------- | ------------------ | ----------- | -------------- |
| **A 轨** | 四川省在线审批平台 | 发改委      | 技术可行性审批 |
| **B 轨** | 国家重大建设项目库 | 发改委+财政 | 纳入预算计划   |

> ⚠️ **关键认知**：完成 A 轨技术审批 ≠ 获得资金，必须同步推进 B 轨（项目库 → 滚动计划 → 年度计划首年）

### 三阶段审批流程

1. **项目建议书**（Stage 1）

   - 成都 2025 年后仅做技术备案，不做投资决策
   - 重心转移到第二阶段

2. **可行性研究报告**（Stage 2）⭐

   - 成都投资决策的关键阶段
   - 6 项前置要件必须办齐
   - 经济评价至关重要

3. **初步设计及概算**（Stage 3）
   - 双重审批：行业主管部门+投资主管部门
   - 概算批复是资金拨付的前提

### 四种融资模式

| 模式           | 适用项目           | 审批特点                 |
| -------------- | ------------------ | ------------------------ |
| **直接投资**   | 非经营性基础设施   | 标准三阶段审批           |
| **资本金注入** | 政府平台公司项目   | 双审批（发改+国资）      |
| **补助**       | 企业投资项目       | 事后申请制               |
| **贴息**       | 需要融资支持的项目 | 二次申请（先项目后贴息） |

## 🎯 五大关键策略

1. **树立"双轨并行"意识** - A 轨+B 轨必须同步推进
2. **熟练使用在线审批平台** - 所有项目的强制性入口
3. **前置要件"并行办理"** - 避免串联延误
4. **适应成都决策后移政策** - 重心从建议书转移到可研
5. **尽早明确融资模式** - 不同模式对应不同路径
