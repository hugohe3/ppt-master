# 设计规范与内容大纲

## 项目基本信息

- **项目名称**：为什么你很努力，却总"记不住"？- 4 个学习误区解析
- **画布格式**：微信公众号头图 (900×383, 2.35:1)
- **设计风格**：通用灵活风格
- **配图数量**：5 张
- **创建日期**：2025 年 10 月 30 日

## 目标受众与使用场景

**受众画像**：

- 职场人士、学生、终身学习者
- 关注个人成长、学习方法论
- 年龄层：18-40 岁
- 追求自我提升，愿意反思学习方法

**使用场景**：

- 微信公众号文章配图（Hugo 的个人成长笔记）
- 手机端阅读为主
- 可能转发至朋友圈或学习社群
- 与正文内容紧密配合，作为视觉锚点

## 设计系统

### 画布规格

- **尺寸**：900×383 px
- **viewBox**：`"0 0 900 383"`
- **宽高比**：2.35:1
- **安全边距**：左右 40px，上下 30px

### 色彩方案

**主色调**：

- **深蓝色**：`#2C3E50` - 主要文字，专业感
- **警示红**：`#E74C3C` - "陷阱"概念的视觉强调
- **活力橙**：`#F39C12` - 重要提示与数据亮点

**辅助色**：

- **柔和灰**：`#7F8C8D` - 次要信息
- **背景浅灰**：`#ECF0F1` - 卡片背景
- **纯白**：`#FFFFFF` - 主背景
- **科学蓝**：`#3498DB` - 科学研究数据展示

**特殊色**：

- **荧光黄**：`#F1C40F` - 模拟高亮笔效果（误区三）
- **温暖米色**：`#F5F5DC` - 笔记纸效果（误区二）

### 字体系统

**字体栈**：

```
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "Helvetica Neue",
             "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif
```

**字号规范**：

- **超大标题**：52-58px（封面主标题）
- **大标题**：38-44px（各误区主标题）
- **中标题**：28-32px（副标题）
- **正文**：22-26px（描述文字）
- **数据强调**：36-40px（关键数据）
- **小字**：18-20px（辅助信息、引用来源）

**字重**：

- 标题：700 (Bold)
- 强调数据：700 (Bold)
- 正文：400 (Regular)
- 次要信息：400 (Regular)

### 布局原则

**横版头图布局** (900×383)：

- **核心内容区**：820×323 px (去除边距后)
- **左右布局**：左侧视觉元素 (380-400px) + 右侧文字 (420-440px)
- **居中布局**：封面使用，单一主题居中对齐
- **垂直居中**：关键元素保持垂直居中
- **信息层次**：标题 → 核心概念 → 支持数据 → 来源

### 图形元素

**图标风格**：

- 简约线性图标，扁平化设计
- 线条粗细：2.5-3.5px
- 圆角处理：柔和边角
- 色彩鲜明，符号化强

**装饰元素**：

- 圆形底板（突出图标）
- 警示符号（叉号、感叹号、问号）
- 箭头、下划线等强调元素
- 渐变背景（仅封面使用）

## 技术规范

### SVG 基础结构

```xml
<svg xmlns="http://www.w3.org/2000/svg"
     viewBox="0 0 900 383"
     width="900"
     height="383">
  <!-- 背景 -->
  <rect width="900" height="383" fill="#FFFFFF"/>

  <!-- 内容层 -->
  <!-- ... -->
</svg>
```

### 文本换行规则

- 使用 `<text>` + `<tspan>` 进行手动换行
- **禁止使用 `<foreignObject>`**
- 每行 `<tspan>` 设置 `x` 和 `dy` 属性
- 行高：字号的 1.4-1.5 倍

### 对齐与间距

- 标题左对齐或居中对齐
- 正文左对齐
- 行高：字号的 1.4 倍（标题）、1.5 倍（正文）
- 标题与正文间距：24-30px
- 段落间距：18-22px
- 元素与边界间距：至少 30px

### 图标设计

- 尺寸：60-80px（主图标）
- 线条粗细：2.5-3.5px
- 背景圆形直径：100-120px
- 圆形透明度：0.1-0.2
- 图标颜色：与主题色一致

## 设计理念

本系列配图旨在：

1. **建立认知冲击**：用科学研究挑战常识，引发读者思考
2. **提供实用价值**：不仅指出问题，更给出替代方案
3. **保持亲和力**：Hugo 的个人经验分享，真诚不说教
4. **便于传播**：每张图独立成章，适合单独使用或组合传播
5. **强化品牌**：统一的视觉风格，体现"理性决策、实际成长"的定位

## 备注

- 本项目为 Hugo 的个人成长笔记公众号文章配图
- 目标是帮助读者识别并避免低效学习方法
- 设计风格专业但亲和，避免说教感
- 强调科学依据（邓洛斯基报告），建立可信度
- 每个误区都有清晰的"为什么低效"解释
