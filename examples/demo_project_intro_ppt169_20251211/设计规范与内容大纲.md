### 设计特点

- **明亮清爽**: 浅色背景，干净舒适
- **现代感强**: 渐变色、圆角、阴影营造层次
- **可读性高**: 高对比度，阅读友好
- **专业友好**: 既有科技感又不冰冷
- **参考风格**: Notion、Linear、Vercel、Stripe

### 布局规范

- **画布尺寸**: 1280×720
- **viewBox**: `0 0 1280 720`
- **边距**: 左右 60px，上下 40px

- **阴影**: `0 4px 12px rgba(0,0,0,0.05)`

## ✅ 设计原则

遵循清新科技风格核心原则：

1. **明亮清爽** - 浅色背景，高对比度文字
2. **现代渐变** - 使用渐变色增加活力
3. **圆角柔和** - 避免尖锐边角，友好感
4. **层次分明** - 通过阴影和颜色区分层级
5. **留白充足** - 内容不拥挤，呼吸感强
