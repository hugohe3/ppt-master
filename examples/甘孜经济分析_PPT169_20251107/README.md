## 核心发现

### 双基准线分析框架

本报告创新性地采用了"双基准线"分析框架：
- **GDP占比基准线**：0.90%（甘孜州GDP占全省比例）
- **人口占比基准线**：1.33%（甘孜州人口占全省比例）
- **效率比值**：0.68（GDP占比/人口占比）

效率比值低于1.0，说明经济产出效率低于全省平均水平32.3%，属于典型的欠发达地区。

### 关键洞察

1. **高转移支付依赖型地区**：财政支出占比（3.38%）是GDP占比（0.90%）的3.76倍，财政自给率仅13.32%

2. **土地财政缺失**：政府性基金收入占比仅0.18%，土地财政能力极弱

3. **债务效率高但空间不足**：债务率仅35.35%，但债务余额已达限额的96%，新增融资空间仅6.5亿元

4. **专项债融资能力严重不足**：专项债限额占比仅0.28%，远低于GDP占比，基建项目融资能力受限

5. **人均公共服务投入全省最高**：人均财政支出是全省平均的2.54倍，体现国家对民族地区的政策倾斜
