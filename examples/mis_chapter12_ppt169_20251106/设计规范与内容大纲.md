## 项目信息

- **项目名称**：MIS 第12章 - 强化决策 (Enhancing Decision Making)

## 设计规范

### 1. 画布规格

```
viewBox="0 0 1280 720"
width="1280" height="720"
```

### 2. 配色方案

采用专业咨询风格配色，以 McKinsey Blue 为主色调：

- **主色（McKinsey Blue）**：`#005587` - 用于标题、关键图表元素
- **辅助蓝色**：`#0076A8` - 用于次级标题、图表辅助元素
- **深灰色**：`#2C3E50` - 用于正文、标注
- **中灰色**：`#7F8C8D` - 用于次要文本、说明
- **浅灰色**：`#ECF0F1` - 用于背景、分隔、卡片底色
- **强调色（橙色）**：`#E67E22` - 用于重点标注、数据高亮
- **成功绿**：`#27AE60` - 用于积极指标
- **背景色**：`#FFFFFF` - 纯白背景

### 3. 字体规范

- **中文字体栈**：
  - 标题：`'Microsoft YaHei', 'PingFang SC', 'Hiragino Sans GB', sans-serif`
  - 正文：`'Microsoft YaHei', 'PingFang SC', sans-serif`

- **英文字体栈**：
  - 标题：`'Segoe UI', 'Arial', sans-serif`
  - 正文：`'Segoe UI', 'Arial', sans-serif`

- **字号标准**：
  - 封面标题：72px（bold）
  - 封面副标题：36px（regular）
  - 页面主标题：48px（bold）
  - 二级标题：32px（bold）
  - 三级标题：24px（semibold）
  - 正文：20px（regular）
  - 小字/注释：16px（regular）

### 4. 版式网格与边距

- **安全边距**：上下左右各 60px
- **内容区域**：1160×600px（居中）
- **网格系统**：12列网格，列间距 20px
- **标题区高度**：100-120px
- **内容区起始**：y=140

### 5. 视觉元素规范

#### 分隔线
- 粗细：2px
- 颜色：`#005587` 或 `#ECF0F1`
- 常用位置：标题下方、区块之间

#### 图标
- 使用简洁的几何图形图标
- 尺寸：32×32px 或 40×40px
- 颜色：与主题配色一致

#### 阴影与层次（咨询风格慎用）
- 仅在必要时使用极浅阴影
- 优先使用留白和色块区分层次

## 文本内容处理原则

### 文字密度控制
- 每页不超过 100 字（中文）
- 使用要点形式，避免长段落
- 关键词加粗或变色

### 层次表达
- 使用编号（1, 2, 3）或符号（•, -, →）
- 缩进 20px 表示子级
- 颜色区分主次（主：`#2C3E50`，次：`#7F8C8D`）

### 双语呈现
- 中文为主，英文为辅
- 专业术语：中文（英文）格式
- 英文字号比中文小 10-15%

## 技术实施要点

### SVG 规范
- 禁止使用 `<foreignObject>`
- 使用 `<tspan>` 手动换行
- 每个 SVG 文件独立完整

### 文本换行
- 标题：单行，超长截断或缩小字号
- 正文：使用 `<tspan>` 手动换行，行距 1.4-1.6
- 列表：每项独立 `<text>` 或 `<tspan>`

### 对齐与网格
- 所有元素严格对齐网格（20px 基准）
- 标题左对齐：x=60
- 内容居中或左对齐
- 使用辅助线确保垂直对齐
